To get a local copy up and running, follow these steps.

### Prerequisites

- A web browser
- A code editor
- A terminal

### Setup

Clone this repository to your desired folder:

```sh
  git clone https://github.com/SergioPeralta22/math-magicians.git
```

Install this project with:

```sh
  cd math-magicians
```
